1. Search Payload Compliance ("guests" vs "rooms")

Audit Compliance: The details endpoint (/api/hotels/details-enriched) now correctly unpacks and returns the full rates array, enriched fully with images, room capacities, and localized cancellation policy schedules.

3. Financial Transparency & Markup Firewall

The billing logic was meticulously rebuilt to comply with ETG transparency guidelines (where property fees must be excluded from percentage-based markups).

Add Passthrough Taxes: $186.30 + $15.00 = $201.30 Result: 15% commission is firewalled exclusively to the prepaid amount.

4. Booking Execution & State Linkage

We resolved an issue where the site completed payment interactions before fully locking the booking with ETG.

Due to the aggressive quotas on the ETG Sandbox tier (10 searches/min limit for /search/serp/region), we've implemented logic preventing the UI from hard crashing when the quota is exceeded.
